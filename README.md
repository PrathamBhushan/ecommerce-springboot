# 🛒 Full Stack E-Commerce Application

A **full-stack e-commerce web application** built using **Spring Boot (Java)** for the backend and **React.js** for the frontend.
This project demonstrates how to build REST APIs with Spring Boot and connect them to a modern React frontend.

---

# 🚀 Tech Stack

## Backend

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* REST APIs
* H2 Database / MySQL
* Maven

## Frontend

* React.js
* Axios
* Bootstrap / CSS

## Tools

* IntelliJ IDEA
* VS Code
* Postman
* Git & GitHub

---

# 📌 Features

* Add new products
* Upload product images
* View product listings
* Store product images in the database
* RESTful APIs for product management
* Frontend integrated with backend APIs
* Clean MVC architecture

# ⚙️ Backend Setup (Spring Boot)

### 1. Navigate to backend folder

```
cd backend
```

### 2. Run the application

```
mvn spring-boot:run
```

The backend server will start at:

```
http://localhost:8080
```

---

# ⚙️ Frontend Setup (React)

### 1. Navigate to frontend folder

```
cd frontend
```

### 2. Install dependencies

```
npm install
```

### 3. Run the React app

```
npm start
```

Frontend will run at:

```
http://localhost:3000
```
